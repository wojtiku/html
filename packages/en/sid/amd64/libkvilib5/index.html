KVIrc (IRC client) base library

A highly configurable graphical IRC client, with powerful and flexible built-in scripting language, supports IRC DCC, DCC over SSL, Spell-checking, drag & drop, file browsing, and much more. KVIrc can be extended using its own scripting language, integrate with KDE, and supports custom add-ons/scripts and themes.

This package contains the main library of KVIrc.
