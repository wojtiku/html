development files for fatrat

FatRat is a feature rich download manager written in C++ and built on top of Qt4 library. It supports a lot of download and file exchange protocols and is continuously extended. It also includes a plugin system.

This package contains FatRat source header files used to compile a plugin or to develop your own one.
